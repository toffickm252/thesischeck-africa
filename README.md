# ThesisCheck Africa

A tool for final-year students in African universities to get
chapter-by-chapter feedback on argument coherence and structural
compliance before thesis submission.

## The Problem

Turnitin checks similarity. Nothing checks if your arguments
hold up or if your structure meets your department's requirements.

## What This Tool Does

Upload your thesis PDF and your department's submission guidelines.
Get feedback chapter by chapter on argument coherence and structural
compliance. Powered by Claude API.

## Stack

- Frontend: React
- Backend: FastAPI (Python)
- AI: Claude API (Anthropic)
